# Marketing Playbooks

1. **Campaign performance** (`mkt-campaign-performance`)
   - Symptom: Performance question
   - Checks: campaign ID, date range
   - Outcome: request report export

2. **Report export** (`mkt-reporting`)
   - Symptom: Need export
   - Checks: report name, format
   - Outcome: request filters

3. **Metric definition** (`mkt-metric-definition`)
   - Symptom: KPI meaning unclear
   - Checks: metric name
   - Outcome: request KB source or escalate

4. **Attribution question** (`mkt-attribution`)
   - Symptom: Attribution unclear
   - Checks: model name
   - Outcome: request model details

5. **Campaign setup** (`mkt-campaign-setup`)
   - Symptom: new campaign
   - Checks: channel and scope
   - Outcome: request requirements

6. **Channel results** (`mkt-campaign-performance`)
   - Symptom: channel results
   - Checks: channel name
   - Outcome: request report export

7. **Report failure** (`mkt-reporting`)
   - Symptom: report failed
   - Checks: error message
   - Outcome: collect details and escalate

8. **Conversion tracking** (`mkt-attribution`)
   - Symptom: tracking question
   - Checks: campaign ID
   - Outcome: request tracking context

9. **KPI mismatch** (`mkt-reporting`)
   - Symptom: KPI mismatch
   - Checks: report filters
   - Outcome: request report export

10. **Compliance reporting** (`mkt-metric-definition`)
    - Symptom: compliance requirement
    - Checks: metric name
    - Outcome: escalate
