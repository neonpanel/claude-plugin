# Marketing — Module Map

## mkt-campaigns

Campaign management and performance overview.

**Common user intents:**
- campaign performance
- campaign setup

**Common confusions:**
- channel vs campaign

**Artifacts to request:**
- campaign_id
- channel_name
- date_range

**Key objects:** Campaign, Channel, Audience

## mkt-analytics

Marketing analytics and reporting.

**Common user intents:**
- kpi meaning
- report export

**Common confusions:**
- metric definition

**Artifacts to request:**
- metric_name
- report_name
- filters

**Key objects:** Metric, Report, Segment

## mkt-attribution

Attribution and conversion tracking.

**Common user intents:**
- attribution question
- conversion tracking

**Common confusions:**
- model differences

**Artifacts to request:**
- model_name
- date_range
- campaign_id

**Key objects:** Attribution Model, Conversion, Touchpoint
