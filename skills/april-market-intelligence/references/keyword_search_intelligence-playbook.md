# Keyword & Search Intelligence Playbooks

## Analyze Search Query Performance (SQP)

1. Export SQP data for the target ASIN(s) — minimum 4-week period.
2. Sort queries by total query volume (highest first).
3. For each high-volume query, calculate:
   - Impression share: your impressions / total query impressions
   - Click share: your clicks / total query clicks
   - Purchase share: your purchases / total query purchases
4. Identify funnel drop-offs:
   - High impression share, low click share → listing problem (image, title, price)
   - High click share, low purchase share → conversion problem (reviews, content, price)
5. Calculate relative CTR: click share ÷ impression share. Below 1.0 = underperforming.
6. Prioritize optimization actions by query volume × gap size.
7. Cross-reference with PPC data to identify queries where paid visibility supplements weak organic.

## Build Keyword Tier Strategy (Brand / Generic / Competitor)

1. Export full keyword list from all campaigns.
2. Categorize each keyword:
   - Brand: contains your brand name or unique product identifiers
   - Competitor: contains competitor brand names
   - Generic: category terms, feature terms, use-case terms
3. Set tier-specific targets:
   - Brand: ACoS 5-10%, high impression share (>80%), defensive priority
   - Generic: ACoS 20-35%, growth focus, keyword expansion
   - Competitor: ACoS 30-50%, conquest focus, monitor conversion rates
4. Create separate campaigns per tier for clean measurement.
5. Allocate budget: typically 10-15% brand, 60-70% generic, 15-25% competitor.
6. Review tier performance weekly and adjust allocation monthly.

## Diagnose Query-Level Funnel

1. Select the target keyword or keyword cluster.
2. Pull SQP data for impression share, click share, cart share, purchase share.
3. Map the funnel stages and identify the biggest drop-off:
   - Impressions → Clicks (CTR stage): driven by main image, title, price, review count
   - Clicks → Add-to-Cart (consideration stage): driven by listing content, A+, reviews, price
   - Add-to-Cart → Purchase (conversion stage): driven by price, shipping speed, stock status
4. For each drop-off, identify the lever:
   - CTR problem → test main image, optimize title, adjust price
   - Consideration problem → improve A+ content, get more reviews, enhance bullet points
   - Conversion problem → check price competitiveness, ensure Prime badge, verify stock
5. Implement changes and re-measure after 2-4 weeks.

## Model Share of Voice

1. Define the keyword universe: top 20-50 keywords by relevance and volume.
2. For each keyword, record:
   - Your organic rank position
   - Your paid position (if advertising)
   - Total search volume estimate
3. Calculate weighted SOV: Σ(visibility score × keyword volume) / Σ(keyword volume).
4. Visibility score: position 1 = 1.0, position 2 = 0.7, position 3 = 0.5, positions 4-10 = 0.3, page 2+ = 0.1.
5. Track SOV weekly and chart the trend.
6. Compare against competitors if data is available.
7. Set SOV targets aligned with revenue goals.
