# Support — Module Map

## sup-access

Access, permissions, and account settings.

**Common user intents:**
- Reset password
- Change permissions
- Account locked out
- Update account settings

**Common confusions:**
- Confusing account-level vs company-level permissions

**Key objects:** User account, Permissions, Login credentials, Access roles

## sup-troubleshoot

General troubleshooting and issue triage.

**Common user intents:**
- Something isn't working
- I got an error
- System is slow
- Data looks wrong

**Common confusions:**
- Not providing error details when reporting issues

**Key objects:** Error message, System status, Debug logs, Issue category

## sup-requests

Support requests and handoffs.

**Common user intents:**
- I need to talk to someone
- Escalate my issue
- Check ticket status
- Request a feature

**Common confusions:**
- Expecting immediate resolution for all escalations

**Key objects:** Support ticket, Escalation path, Priority level, Resolution status

## sup-feature-guidance

Feature usage guidance, onboarding walkthroughs, dashboard configuration, and product returns workflows. Covers how to use specific product features, initial setup steps, and navigating key UI areas.

**Common user intents:**
- How do I use this feature?
- Walk me through onboarding
- How do I configure my dashboard?
- How do product returns work?
- Where do I find this setting?

**Common confusions:**
- Confusing feature configuration with account settings
- Not understanding that dashboard layout is customizable
- Mixing up product returns with refunds in accounting

**Key objects:** Feature walkthrough, Onboarding checklist, Dashboard settings, Product returns, UI navigation
